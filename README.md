# Esempio Angular 20 + PrimeNG 20 (IT)

Progetto minimale solo front‑end che replica una schermata con **p-table** e **p-dialog**.

## Requisiti
- Node.js 20+ (o 22/24)  
- TypeScript 5.9 (installato come dev dependency)
- Angular CLI v20

## Avvio
```bash
npm install
npm start
```
Poi apri http://localhost:4200/

## Note
- Tema: Aura (v20 – via `@primeuix/themes`) 
- Lingua: it‑IT (LOCALE_ID + `registerLocaleData`)
- Componenti usati: Table, Dialog, Button, InputText, Dropdown, Tag, Tooltip
